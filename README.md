# JSON教程
# GitHub上README.md排版样式教程
## 标题
     在文本下面加上 等于号 = ，那么上方的文本就变成了大标题
     文本下面加上 下划线 - ，那么上方的文本就变成了中标题
## 显示文本
### 普通文本
     换行需要使用<br>(或者<br/>)
### 单行文本
    前面有两个Tab
### 多行文本
    在每行行首加两个Tab
### 部分文字的高亮
    用 `  ` 包围起来。注意这不是单引号，而是Tab上方
### 文字超链接
    [ 要显示的文字 ]( 链接的地址 )
 [百度](http://www.baidu.com)
## 插入符号
### 圆点符
    编辑的时候使用的是星号 *,*号后要有空格，二级圆点和三级圆点。就是多加一个Tab
   * list1
   * list2
## 缩进
>数据结构  
>>树  
>>>二叉树
